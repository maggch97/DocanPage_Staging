Стандартные UV чернила

Подходят для всех видов жестких материалов, яркие цвета, отличная адгезия и устойчивость к внешним воздействиям

Гибкие UV чернила

Специально разработаны для гибких материалов, обладают эластичностью, не трескаются, идеальны для изгиба и растяжения

Белые UV чернила

Чернила с высокой укрывистостью, подходят для прозрачных и темных материалов, обеспечивают яркую подложку

UV лак

Обеспечивает глянцевый или матовый эффект, улучшает текстуру и защиту отпечатков

Металлические чернила

Создают эффект золота, серебра, меди и других металлов, идеальны для премиальной упаковки и декора

Флуоресцентные чернила

Светятся под ультрафиолетом, подходят для защитных маркировок и специального декора

Защитные чернила

Содержат специальные маркеры, могут использоваться для защиты продукции и отслеживания

Набор CMYK

Базовый набор из четырех цветов, подходит для большинства стандартных задач печати

Набор CMYK + белый

Набор из пяти цветов с белыми чернилами, идеален для печати на прозрачных и темных материалах

Набор CMYK + белый + лак

Премиальный набор из шести цветов, обеспечивает богатые визуальные и тактильные эффекты

Полный набор цветов

Профессиональный набор, включающий все цвета, для премиальных задач печати

Очищающая жидкость

Профессиональная жидкость для очистки печатающих головок, эффективно удаляет остатки чернил, продлевает срок службы

Набор для очистки

Комплексный набор для обслуживания принтера, включает все необходимые инструменты для очистки

Интеллектуальная система печатающих головок

Система UV LED отверждения

  • Двойные пневматические валы для удобной загрузки
